*環境構築*

Dockerビルド

　1.git clone リンク

　2.docker-compose up -d build

Laravel環境構築
　
　1.docker-compose exec php bash
  
　2.composer install
  
　3..env.exampleファイルから.envを作成し、環境変数を変更

　4.php artisan key:generate

　5.php artisan migrate

　6.phpartisan db:seed

*使用技術*

　・Laravel 10.0
